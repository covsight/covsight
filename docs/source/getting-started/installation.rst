############
Installation
############

Requirements
============

* Python 3.10 or later
* Linux, macOS, or Windows

Basic Install
=============

.. code-block:: bash

    pip install covsight

This installs the ``covsight`` command-line tool and all core library functionality.

Verify the Install
==================

.. code-block:: bash

    covsight --help

You should see the list of available sub-commands.
