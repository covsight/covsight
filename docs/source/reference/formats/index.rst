################
File Formats
################

Technical specifications for the file formats read and written by covsight.

The primary storage format is **NCDB** (``.cdb``), which is documented in
detail in the ``covsight-core`` package. This section covers the interchange
and input formats that the ``covsight`` CLI can read and convert.
